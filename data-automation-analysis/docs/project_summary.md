# Project Summary

This project analyzes U.S. flight delay data to identify patterns in airline performance and route efficiency. The primary objective was to clean a large dataset, reduce noise, and extract meaningful insights through data analysis and visualization.

The dataset was processed using Python and Pandas, where missing values, redundant columns, and inconsistencies were addressed. After cleaning, the dataset was reduced to essential variables including airline, route, delay metrics, and distance.

The analysis revealed that delays are influenced more by operational factors and route-specific conditions rather than flight distance alone. Certain airlines and routes consistently exhibited higher delays, indicating potential inefficiencies.

Visualizations were created to communicate these findings effectively, including airline performance comparisons, route delay analysis, and delay distribution patterns.

This project demonstrates practical data analysis skills, including data cleaning, transformation, aggregation, and visualization, applied to a real-world dataset.
